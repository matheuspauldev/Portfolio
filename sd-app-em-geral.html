Projetos Destacados


1. Aplicativo de Saúde Mental

Desafio: Desenvolver um aplicativo móvel para auxiliar pessoas com ansiedade e depressão, oferecendo recursos de acompanhamento de humor, meditações guiadas e conexão com terapeutas.

Solução: Criei um aplicativo React Native com as seguintes características:

  • · Interface calma e acolhedora com animações suaves usando Lottie
  • · Integração com HealthKit e Google Fit para monitoramento de atividades
  • · Chat criptografado para comunicação segura com terapeutas
  • · Sistema de notificações personalizadas baseado em padrões de humor

Tecnologias: React Native, Redux, Firebase, Node.js, WebRTC

Resultado:

  • · 50.000+ downloads na App Store e Google Play
  • · Aumento de 70% na adesão a tratamentos de saúde mental

2. Dashboard Analítico para E-commerce

Desafio: Criar uma plataforma web robusta para análise de dados de e-commerce em tempo real, capaz de processar e visualizar grandes volumes de dados.

Solução: Desenvolvi uma aplicação web em React com:

  • · Gráficos interativos e personalizáveis usando D3.js e Recharts
  • · Implementação de virtualização para listas longas de produtos
  • · Sistema de filtros avançados com query builder visual
  • · Exportação de relatórios em múltiplos formatos

Tecnologias: React, TypeScript, GraphQL, Apollo Client, Node.js, PostgreSQL

Resultado:

  • · Redução de 20% no tempo de tomada de decisão para gestores de e-commerce
  • · Capacidade de processar diversas transações diárias em tempo real
